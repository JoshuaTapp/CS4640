Cascading Style Sheets - CSS

CSS is a language used to describe presentation of a web page

  • Designed to separate concerns, the visual representation (CSS) from the structure (HTML) to increase readability and maintainability
    • HTML: content and structure
    • CSS: style and layout
    • This leads to reusing style across elements and pages
    • update HTML without needing to update styles
  • 3 ways to add CSS to HTML
    1. Inline
      1. only applies to that element. Its in the HTML itself.
    1. Document Level
      1. Add to HTML doc and it applies to that page
      1. uses the <style> tag block to add CSS inside <head> block.
    1. External Level
      1. import CSS and use across pages
      1. create style.css file with all your CSS. In <head> add a <link> element with a reference to the CSS file
      <head>
	<link rel="stylesheet"
				type="text/css"
				href="style.css" />
</head>
  • Structure
    • selector: a pattern used to select which HTML element(s) are styled
      • can have multiple declarations per selector
      • can use multiple selectors with a declaration block
    • declaration: how to style selected elements
  • Properties
    • Box Props
  • Selectors
    • Element: HTML element
      • h1, p, div { bla bla bla }
    • ID: refers to the single (unique) element with the matching attribute id="ID-example"
      • #intro { bla bla bla }
    • Class: select multiple elements with a given class name
      <p class="warning">Warning text goes here.</p>
p.warning {color: red; font-weight: bold; }
    • Generic Class: Class but with any type
      <p class="warning">Warning text goes here.</p>
.warning {color: red; font-weight: bold; }
    • Pseudo-class: selects elements when they are in a particular state.
    • Universal: select all elm in the doc or all elements inside another elm.
      • Every elm in HTML:
      * { color: blue; font-size: 12px; }
      • Every element inside <div> elm:
      div * { color: green; font-size: 11px; }
    • Attribute: select elms based on the value of an attribute
    • Combining Selectors Pseudo-Element
      • Descendant: space separated
        • any child in the tree will be selected
        • ex: any span inside of a div at any level
        div span { color: green; font-size: 11px; }
      • Direct Child: separated by >
        • only direct children selected
        • div > p { bla bla }
      • Adjacent Sibling: separated by +
        • the sibling directly below the first arg
        li.selected + li { bla bla bla }

<ul>
	<li>First</li>
	<li class=“selected”>Second</li> 
	<li>Third</li> /* THIS ONE IS SELECTED */
	<li>Fourth</li>
</ul>
      • General Sibling: separated by ~
        • all next siblings
        li.selected ~ li { bla bla bla }

<ul>
	<li>First</li>
	<li class=“selected”>Second</li> 
	<li>Third</li>  /* THIS ONE IS SELECTED 
	<li>Fourth</li> AND THIS ONE */
</ul>
